# Javi's Board

A personal Kanban board — mobile-friendly, offline-capable, zero dependencies.

## Features
- 5 columns: Backlog → To Do → In Progress → Review → Done
- Drag & drop (desktop + touch/mobile)
- Editable categories with custom colours (⚙️ button)
- Priority tags (High / Medium / Low)
- Due dates with overdue highlighting
- Persists in browser localStorage
- Filter by category

## Live
Hosted on GitHub Pages → bookmark on your phone.

## Usage
Open `index.html` in any browser, or visit the GitHub Pages URL.
Keyboard shortcut: `N` = new task, `Esc` = close modal.
